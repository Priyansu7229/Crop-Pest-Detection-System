# Crop Pest Detection using YOLOv8

This project uses YOLOv8 to detect multiple crop pests (like ants, beetles, moths, etc.) from images, videos, or live webcam feed.

Features
- Trained YOLOv8 model on custom insect dataset
- Works with images, videos, or live webcam
- Built using Streamlit for easy UI
- Deployed locally or on Streamlit Cloud

Model Info
- Model: YOLOv8n (Ultralytics)
- Classes: 12 insect types
- Framework: PyTorch
